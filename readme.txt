== Description ==

Replaces the default text widget with one that allows Javascript so you can do basic things like add Google Ads to your sidebar without using other plugins.

Important: It's only intended for Multisite. If you use it in standard WordPress and it doesn't work or breaks something don't expect any sympathy.

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in Network Admin
